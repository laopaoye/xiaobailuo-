小白螺是一款功能齐全的AI内容创作工具，专为现代影视与短剧创作者设计。通过集成最新的生成式AI模型，它可以帮助您：

- 📝 **智能剧本创作** – 基于剧情梗概生成专业剧本、对话和分镜描述
- 🎨 **角色与场景设计** – 自动生成角色形象、多视角、多机位、场景概念图、道具图，360度全景图、融图等功能，
- 🎬 **AI视频生成** – 参考图生视频/首尾帧生成视频/文生视频（豆包seedance2.0、豆包seedance1.5、HappyHorse、可灵、PixVerse、VIdu、通义万相、）
- 🗣️ **语音合成** – 多角色、多情绪的AI配音
- 🎵 **背景音乐生成** – 根据情绪自动生成配乐
- ✍️ **剪辑与后期** – 并支持导出专业剪辑格式

## ✨ 主要功能

- **剧情模式**：将脚本自动转化为剧情漫剧或写实真人剧（图像+配音+运镜）
- **解说模式**：生成爆火解说模式
- **商业片辅助**：产品展示、宣传片脚本+分镜+配乐一键生成
- **全链路工作流**：从创意到成品，只需一个Web界面
